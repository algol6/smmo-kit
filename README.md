# smmo-kit

Discord Bot for SMMO

---
## Configuration

Create .env file with following value:
```
DISCORD_TOKEN = "discord-token-here"
SMMO_TOKEN = "smmo-token-here"
```

---
## Installation

### MacOS / Linux
`python -m venv .venv`

`source venv/bin/activate`

`pip install -r requirements.txt`

### Windows
`python -m venv venv`

`./venv/bin/activate.bat`

`pip install -r requirements.txt`

## Running

`python run.py`
